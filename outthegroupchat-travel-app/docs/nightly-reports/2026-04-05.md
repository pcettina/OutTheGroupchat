## Known Patterns Observed

- **Zod error format change:** L2 changed validation error responses from `{ success: false, error: '...' }` to `{ error: 'Validation failed', details: issues }`. Any test asserting `body.success === false` on a 400 Zod response will break. Pattern now documented in MEMORY.md.
- **Rate-limit leak audit:** 6 test files were missing the rate-limit mock this build. Still need to audit all new test files on each build.
- **aiLogger missing from setup.ts:** Routes that import `aiLogger` from `@/lib/logger` fail silently if setup.ts doesn't mock it — Zod validation 400 becomes 500. Now fixed.
