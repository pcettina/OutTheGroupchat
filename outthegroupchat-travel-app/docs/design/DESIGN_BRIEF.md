# Design Brief

> **Status:** Not yet populated. Run `/design-research` to generate.

This document will contain the locked-in visual and tonal foundation that every later design session references. It's the single source of truth for:

- Competitor landscape synthesis (5–10 apps in the same space, with screenshots + extracted patterns)
- **Locked color palette** (3 primary + 2 accent + neutral scale, in hex)
- **Type pairing** (display + body + optional mono/accent, with licensing notes)
- **Tone / voice guide** (do's, don'ts, 10+ sample copy strings for real OTG surfaces)
- **Motion language** (timing values, easing curves, when to use haptics)
- **Visual mood-board** (3–5 reference styles, named — e.g., "neo-brutalist night-out," "y2k diner")
- **Design principles** (5–7 short rules — e.g., "Every screen has one primary action")
